# Documentation Agent

## 概要
自然言語から要件定義書を自動生成するAPIアプリケーション。  
プロダクト開発初期における要件整理を効率化することを目的としています。

---

## 背景
要件定義は以下の課題を持つ：

- 属人化しやすい
- 初期設計のボトルネックになる
- ドキュメント作成に時間がかかる

本プロジェクトでは、LLMを活用しこれを自動化。

---

## 技術スタック

- FastAPI
- OpenAI API
- Python
- dotenv

---

## 機能

- 自然言語から要件定義生成
- Markdown形式で出力
- 機能一覧・非機能要件・リスクなどを自動構造化
